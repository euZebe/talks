### Problèmes * méthode intestable * objectif de la méthode difficile à identifier * que fait-elle après le persist ? Probablement plus la création d'une transaction ### Préconisations * On trouve souvent la préco de méthodes de 10 lignes max * meilleure testabilité * permet le redécoupage du code en méthodes à responsabilité simple * avoir éventuellement une méthode orchestrant les différentes étapes * meilleur nommage des méthodes (plus simple si elles font peu de choses) * pas une règle absolue ; exemple switch-case
``` public Payment validatePayment(String paymentId) throws InputException { log.info("validating payment " + paymentId); PaymentImpl payment = getPayment(paymentId); if (payment != null) { switch (payment.getStatus()) { case PENDING: payment.setStatus(Status.COMPLETED); if (payment.getAgreement().isDelayedCapture()) { payment.getTransactions().iterator().next() .setStatus(com.lyra.ecs.core.model.Transaction.Status.COMPLETED); } else { payment.getTransactions().iterator().next() .setStatus(com.lyra.ecs.core.model.Transaction.Status.CAPTURED); } try { paymentDao.save(payment); } catch (Exception e) { log.error(e.getMessage(), e); throw new InputException(e.getMessage(), e); } break; default: InputException e = new InputException( "cannot validate a payment with status " + payment.getStatus()); log.error(e.getMessage(), e); throw e; } } else { throw new InputException("payment with uuid = " + paymentId + " is null"); } log.info("payment " + payment.getUuid() + " valid with status = " + payment.getStatus()); return payment; } ``` Note: * complexité cyclomatique (imbrication d'accolades)
``` public Payment validatePayment(String paymentId) throws WebpayInputException { log.info("Transbank payapp - validating payment " + paymentId); // Find & check payment PaymentImpl payment = getPayment(paymentId); if (payment == null) { throw new WebpayInputException("payment with uuid = " + paymentId + " is null"); // fail fast. } switch (payment.getStatus()) { case PENDING: markTransactionToNextStepThenSave(payment); break; default: WebpayInputException e = new WebpayInputException( "cannot validate a payment with status " + payment.getStatus()); log.error(e.getMessage(), e); throw e; } log.info("Transbank payapp - payment " + payment.getUuid() + " valid with status = " + payment.getStatus()); return payment; } private void markTransactionToNextStepThenSave(PaymentImpl payment) throws WebpayInputException { payment.setStatus(Status.COMPLETED); TransactionImpl transaction = payment.getTransactions().iterator().next(); transaction.setStatus( // payment.getAgreement().isDelayedCapture() // ? com.lyra.ecs.core.model.Transaction.Status.COMPLETED // : com.lyra.ecs.core.model.Transaction.Status.CAPTURED // ); try { paymentDao.save(payment); } catch (Exception e) { log.error(e.getMessage(), e); throw new WebpayInputException(e.getMessage(), e); } } ``` Note: * fail fast * retours à la ligne avec // pour meilleur lisibilité * alt + shift + M est ton ami (pour extraire une méthode)
